What Munaciello Does

Six guarantees.

Adversarial Review

Every change gets reviewed by up to 3 different AI models — GPT, Gemini, Claude — each trying to find bugs, security holes, and logic errors. They disagree with each other. That's the point.

SQL Verification Ledger

Every build, test, and lint result is INSERTed into a SQLite table. The evidence bundle at the end is a SELECT, not prose. If the INSERT didn't happen, the verification didn't happen.

Baseline Snapshots

Before touching anything, Munaciello records your project's state — build, diagnostics, tests. After changes, it compares. Regressions are caught before you see the code, not after you ship it.

Pushback

Munaciello is a senior engineer, not an order taker. Bad idea? Tech debt? Dangerous edge case? It tells you before writing a single line. You can override it. But you'll know.

Session Memory

Broke a file last session? Munaciello remembers. Learned your build command? Won't ask again. Context that actually persists across conversations.

Git Autopilot

Before writing code, Munaciello checks your git state — stashes uncommitted work, creates a branch off mainso you're never editing trunk directly. After verification passes, it commits with a clear message and gives you a one-line rollback.

Under The Hood

Every task runs through the same loop.

  1. 1
    Understand & Recall

    Parses your request. Checks session history for past work on these files, including what broke.

  2. 2
    Survey & Pushback

    Searches for existing patterns to reuse. Tells you if the approach is wrong before writing code.

  3. 3
    Baseline

    Snapshots build, tests, diagnostics into the SQL ledger. This is the "before" photo.

  4. 4
    Implement

    Makes changes following your codebase patterns. Extends what's there instead of inventing new abstractions.

  5. 5
    The Hauntverification

    Build, type check, lint, test — all recorded. Then up to 3 AI models attack the code. Issues get fixed before you see anything.

  6. 6
    Evidence & Commit

    Structured report from SQL. Auto-commit with rollback command. Done.
